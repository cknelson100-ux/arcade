# Arcade

Two single-file browser games. Hosted via GitHub Pages.

- **slay.html** — deckbuilder roguelike (15-floor branching map, drag-to-target cards)
- **euchre.html** — 4-player euchre vs. 3 bots (stick-the-dealer, going alone, 4 ambient soundtracks)

Both are self-contained: zero build step, zero external dependencies beyond Google Fonts.

## Play

Open `index.html` for the menu, or jump straight into a game. On iOS / Android, add to home screen for a fullscreen app.

## Files

| File          | What                                                   |
|---------------|--------------------------------------------------------|
| `index.html`  | Landing page with both games                           |
| `slay.html`   | Slay roguelike                                         |
| `euchre.html` | Euchre card game                                       |
| `manifest.json` | PWA manifest (home-screen install)                   |
| `icon*.svg`   | App icons                                              |
